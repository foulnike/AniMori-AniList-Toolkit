# История изменений

## [1.7.4]

Первый релиз на GitHub.

### Изменено
- Репозиторий переименован в `AniMori-AniList-Toolkit`; обновлён адрес загрузки словаря перевода.
- Словарь интерфейса реструктурирован по категориям.

### Возможности (на момент релиза)
- Перевод интерфейса AniList по словарю; русские названия, описания, персонажи и персонал с Shikimori.
- Встроенный аниме-плеер, рейтинги MAL и Shikimori, дерево франшизы, опенинги/эндинги.
- Внешние ссылки (RuTracker, YummyAnime, AnimeGO, MangaLib), русский поиск.
- Экспорт списка из Shikimori в AniList.
- Локальный кэш (IndexedDB), гибкие настройки модулей, логгер.
